# 📊 HotelRevAI – AI Driven Revenue Analysis for Hotels

## 🚀 Overview

This project is part of the Infosys Springboard training program focused on building an AI-driven revenue analysis system for the hospitality industry.

The goal is simple but powerful:
👉 analyze hotel data
👉 identify revenue patterns
👉 suggest smarter pricing & business strategies

Basically, turning raw hotel data into money-making insights.

## 🎯 Project Objective

* Understand hotel booking behavior
* Analyze occupancy, revenue, and demand trends
* Build dashboards and insights
* Apply AI/analytics to improve revenue decisions

---

## 🧠 Tech Stack

* SQL (Data Extraction & Analysis)
* Excel / Power BI (Dashboarding)
* Python (Data Analysis & AI Models)
* DAX (KPIs & Measures)

---

## 📊 Key Metrics Explained

* **Occupancy Rate** → % of rooms booked
* **ADR (Average Daily Rate)** → revenue per room sold
* **RevPAR (Revenue per Available Room)** → overall performance metric

---

## 💡 Key Insights (Update as you progress)

* Weekends show higher occupancy
* Certain cities generate more revenue
* Pricing strategy impacts booking behavior
